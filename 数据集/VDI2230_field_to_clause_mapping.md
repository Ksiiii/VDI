# 字段—VDI 2230 条款/公式映射表（严格版数据集）

> 说明：本表将生成数据集中的列，与 VDI 2230 的核心条款/公式建立对应关系，方便审计与追溯。章节/公式号以 VDI 2230-1 为准（中文译本章节号可能与德/英版略有排版差异）。

## 1. 基础几何/材料
- `series`：公制粗牙系列（用于查表 `d`, `P`, `d2`, `As`） — 参考标准的螺纹表（附录/标准件表）  
- `d_mm` / `P_mm` / `d2_mm` / `As_mm2`：螺纹公称直径/螺距/中径/抗拉应力面积 — 螺纹参数表  
- `class`：强度等级（8.8/10.9/12.9），用于定义证明强度与抗拉强度 — 强度等级表  
- `Eb_MPa`：螺栓材料弹性模量 — 材料参数表  
- `Ec_MPa`：被夹紧件等效弹性模量 — 材料参数表/工程取值

## 2. 力矩—预紧力关系（装配阶段）
- `MA_Nm` ↔ `FM_N`：**R13/1 风格经验式**  
  \( M_A = F_M \left( \tfrac{P}{2\pi} + 0.16\,\mu_G\,d_2 + 0.58\,\mu_K\,D_K \right) \)  
  对应：装配扭矩的经验公式、A1–A5 的摩擦系数选区（螺纹 \(\mu_G\)、承压面 \(\mu_K\)），承压有效直径 \(D_K\)。  
- `muG` / `muK`：摩擦系数（对应 A5/A1–A4 中的范围/等级）  
- `Dk_mm`：承压有效直径（螺栓头/垫圈）

## 3. 回弹与载荷分配（工作阶段）
- `deltaS_mm_per_N`：螺栓回弹 \(\delta_S = L_b/(A_s E_b)\) — 章节：弹簧模型/回弹定义（S 侧）  
- `deltaP_mm_per_N`：**被夹紧件回弹（5.1.2 圆锥/截头圆锥模型）**  
  等效表达：\( \delta_P \approx \frac{4L_k}{\pi E D_K D_2} \)，其中 \( D_2 = D_K + 2\tan\alpha\,L_k \)，\( \alpha \approx 30^\circ \)  
- `Phi`：**R3/3 思路**：\( \Phi = \frac{n\delta_P}{\delta_S + n\delta_P} \)，本脚本默认同轴 \(n=1\)。若有偏心/开口，需接 5.2.2/5.3.x 计算 \(n\) 与上限。

## 4. 外载与残余夹紧力
- `FA_N`：外载（本脚本按证明载荷比例采样，用于示例/训练）  
- `Fb_max_N`：螺栓最大轴力 \(F_{b,\max} = F_M + \Phi F_A\) — 与 3/5 的 \(F_{SA}\) 表述等价  
- `Fk_residual_N`：残余夹紧力（考虑嵌入/松弛），\(F_{KR} = F_M - (1-\Phi)F_A - \Delta F_{Vth}'\) — 3/6、R4 框架

## 5. 预紧力损失与最小装配预紧力
- `deltaZ_mm`：嵌入/温差等效位移（本脚本 3–20 μm 随机）  
- `ΔF` 换算：\( \Delta F = \Delta Z/(\delta_S + \delta_P) \) — **R4**
- `F_proof_N`：证明载荷（由证明强度与 \(A_s\) 得到）  
- `FM_N`：装配预紧力（扰动版由扭矩反算；理想版为目标预紧力）  
- `FM_min`（中间量，未单独存储）：\(F_{M,\min} = F_{Kerf} + \Phi F_A + \Delta F_{Vth}'\) — **R5/1**  
  本脚本示例未给定 \(F_{Kerf}\)，按 0 处理；装配目标取 \(\max(FM_{nom}, FM_{min})\)，并施加 \(\le 0.9 F_{\text{proof}}\) 的装配上限。

## 6. 强度与通过判据
- `sigma_b_MPa`：轴向应力 \(\sigma = F_{b,\max}/A_s\)  
- `SF_yield` / `SF_ult`：屈服/极限安全系数  
- `pass_fail` / `fail_reason`：本脚本默认判据：\(Fk_{residual} > 0\) 且 \(SF_{yield} \ge 1.2\)。可按贵司标准调整。

## 7. 可扩展（未在本脚本中默认启用）
- 偏心/多螺栓分配（5.2.2）→ 计算载荷系数 \(n\) 并更新 \(\Phi\)  
- 开口连接（5.3.x）→ 使用上限曲线处理抬离后的接触/载荷迁移  
- 接触压力、剪切/防滑、密封等（R10、R12…）按需加入为额外标签
