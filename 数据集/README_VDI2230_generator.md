# VDI2230 严格版数据生成器使用说明

## 快速使用
```bash
python vdi2230_generator_strict.py --ideal_rows 10000 --perturbed_rows 10000 --seed 123
```
会在当前目录生成：
- `VDI2230_vdiStrict_ideal_10000.csv`
- `VDI2230_vdiStrict_perturbed_10000.csv`

## 参数说明
- `--ideal_rows`：理想装配样本数（直接按 R13/1 正向求扭矩得到）
- `--perturbed_rows`：扰动样本数（基于名义扭矩+摩擦小扰动后反算预紧力）
- `--per_base`：每个基样本的扰动份数，默认 2（1 份名义 + 1 份扰动）
- `--seed`：随机种子，确保可复现

## 模型与条款对应（摘要）
- R13/1：装配扭矩经验式（0.16/0.58 + μ_G/μ_K + d2/Dk）
- 5.1.2：被夹紧件回弹的圆锥/截头圆锥模型（本脚本给出等效表达）
- R3/3：Φ 由 δS/δP 计算得到（本脚本默认为同轴 n=1）
- R4：嵌入/温差等换算为 ΔF
- R5：最小装配预紧力合成（本脚本示例未提供 F_Kerf，按 0 处理）

## 标定建议
- 用少量实测点回归 μ_G/μ_K（按 A1–A5 的等级范围作先验）与嵌入量 ΔZ 的分布。
- 若场景涉及偏心/开口：接入 5.2.2/5.3.x 求 n 与上限关系。
